Default value. No shadow is displayed h-offset Required. The horizontal offset of the shadow. A positive value puts the shadow on the right side of the box, a negative value puts the shadow on the left side of the box v-offset Required. The vertical offset of the shadow. A positive value puts the shadow below the box, a negative value puts the shadow above the box blur Optional. The blur radius. The higher the number, the more blurred the shadow will be spread Optional. The spread radius. A positive value increases the size of the shadow, a negative value decreases the size of the shadow color Optional. The color of the shadow. The default value is the text color. Look at CSS Color Values for a complete list of possible color values.
